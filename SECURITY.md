# Security Policy

## Supported Versions

Only the latest version of Aura Finance is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 2.x     | :white_check_mark: |
| < 2.0   | :x:                |

## Reporting a Vulnerability

We take the security of Aura Finance seriously. If you believe you have found a security vulnerability, please report it to us by following these steps:

1. **Do not open a public issue.**
2. Email your report to `security@aura-finance.vercel.app` (replace with your actual security contact if available).
3. Include a detailed description of the vulnerability and steps to reproduce it.

We will acknowledge your report within 48 hours and provide a timeline for a fix.

Thank you for helping keep Aura Finance secure!
