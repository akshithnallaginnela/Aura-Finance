# Contributing to Aura Finance

First off, thank you for considering contributing to Aura Finance! It's people like you that make Aura Finance such a great tool.

## Code of Conduct
By participating in this project, you are expected to uphold our Code of Conduct.

## How Can I Contribute?

### Reporting Bugs
* **Check the existing issues** to see if the bug has already been reported.
* If you can't find an open issue that describes the problem, **open a new one**.
* Include as much detail as possible, including steps to reproduce, what you expected to happen, and what actually happened.

### Suggesting Enhancements
* **Open a new issue** with a clear title and description.
* Explain why this enhancement would be useful to most Aura Finance users.

### Pull Requests
* **Fork the repository** and create your branch from `master`.
* If you've added code that should be tested, add tests.
* Ensure the test suite passes.
* Make sure your code lints.

## Development Workflow

### Frontend
1. Install dependencies: `npm install`
2. Start development server: `npm run dev`

### Backend
1. Install dependencies: `pip install -r backend/requirements.txt`
2. Start backend server: `python backend/app.py`

## Style Guide
* **TypeScript**: Use functional components and hooks.
* **Python**: Follow PEP 8.
* **CSS**: Follow the existing glassmorphic variable-based system.

Thank you for your contributions!
